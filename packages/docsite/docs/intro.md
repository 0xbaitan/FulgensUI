# Introduction to FulgensUI

Welcome to **FulgensUI** - a modern UI Component Library built with React and PandaCSS for reusability in frontend projects.

## What is FulgensUI?

FulgensUI is a comprehensive component library that provides:

- 🎨 **Beautiful Components** - Pre-styled, customizable UI components
- ⚡ **PandaCSS** - Zero-runtime CSS-in-JS with type-safety
- 📦 **Modular** - Import only what you need
- 🔧 **Customizable** - Full control over styling and behavior
- 📖 **Well Documented** - Extensive documentation and examples

## Getting Started

Install FulgensUI in your project:

```bash
npm install @fulgensui/core
```

Import and use components:

```jsx
import { Button } from '@fulgensui/core';

function App() {
  return <Button variant="primary">Click me!</Button>;
}
```

Browse our component library in the **Components** section to see all available components with examples and API documentation.

## Development

This library is built using:

- **Vite** - Fast build tool
- **React** - UI library
- **PandaCSS** - Styling solution
- **Storybook** - Component development environment
- **Turborepo** - Monorepo build system
